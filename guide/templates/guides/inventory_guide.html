Locations

Introduction

Creating storage locations for your inventory is not a necessity but we highly recommend it before adding any of your inventory items. Simply click “Locations” under “Inventory” on the menu and select “Create New Location”. You may then add the name of your location and also include temperature or short description of the location. Of course, you are able to change the name of the location should you wish to do so later on.

Reagents

Introduction

Reagents are necessary for creating your assays and extraction protocols. To add a reagent to the system, click on “Inventory” on the menu and select “Reagents”. Then click on “Add New Reagent” you will see there are several fields to fill. Do not worry, we will walk you through this by first adding reagents from a common extraction kit made by Qiagen. Then adding generic reagents that are most commonly used in PCR assays.

  • Step 1 - The QIAmp Fast DNA Stool Mini Kit (LINK HERE) (Cat#51604) contains 6 reagents but requires ethanol (96-100%) by the user.
  • Step 2 - Knowing this, let us first lets add InhibitEX Buffer. To do so, click on “Inventory” and select “Reagents”. Click on “Add New Reagent”, set the Name as “InhibitEX Buffer”, Brand as Qiagen, and the Lot Number on the container. If you do have the item on hand then enter the lot number shown otherwise simply set the lot number as “L12341234”.
  • Step 3 - For the Catalog Number set to “51604” and for the usage set as “EXTRACTION”. Set the Volume Storage and Volume Unit accordingly to what you currently have. In this case, the kit contains 140 milliliters of InhibitEx Buffer. So we will set the volume as “140” and the volume unit as “mL”.
  • Step 4 - Leave the remaining fields empty (as they are only relevant for reagents used in PCR) except for the Expiration Date and Location. Keep in mind that these two fields are optional although it is highly recommended for time-sensitive products to have an assigned expiration date as well as choosing a storage location to encourage organization in your lab.
  • Step 5 - In this case, we will leave the Expiration Date empty but set the location to “Extraction Reagents Room”. Note that we will be assigning this location to all extraction reagents as a demonstration. Ideally, each of these reagents require different storage conditions.
  • Step 6 - Once everything has been confirmed, click “Add Reagent”. Be aware that by doing so, you will no longer be able to modify this reagent except for its Volume Stored, Volume Unit, Expiration Date, and Location.
  • Step 7 - We will do this exact same process for Buffer AL, Buffer ATE, Proteinase K, and 1L of ethanol. The ethanol will have the Name “Ethanol”, Brand as “Generic”, and Catalog Number as “C12341234”.
  • Step 8 - Once we get to adding Buffer AW1 and Buffer AW2, you will notice how these reagents are supplied as concentrates. Keep in mind that Qiagen instructs you to add 25 mL of ethanol to the concentrate volume of 19 mL. So when adding this reagent to the system, set the Volume Stored as the total volume of 44 mL. For Buffer AW2, we will set the Volume Stored as the total volume of 43 mL.
  • Step 9 - In the reagents page, set Usage to “EXTRACTION” and you should see that your list of reagents should look similar to the following image.
